# Zadání:

> - Použijte HTML strukturu stránku s profilem herce **Rowana Atkinsona** na zdrojove adrese vzor [vzoru](https://codepen.io/SimonB87/pen/gOpdZpY) nebo použijte Váš výsledek z předchozího cvičení.
> - Kód zkopírujte do svého vlastního nového HTML souboru.
>
> **Proveďte následující úpravy**:
>
> - Obrázku přiřaďte id: `"profilePicture"`
> - Obrázku přiřaďte třídu: `"round"`
> - Vytvořte si na disku soubor CSS a nalinkujte ho k vašemu html souboru
> - Vytvořte v tomto CSS souboru pravidlo pro id: `"profilePicture"` šířku 300px a hranici elementu o šířce 10px tmavě modré barvy:
>   - `width: 300px;`
>   - `border: 10px solid darkblue;`
> - Vytvořte v tomto CSS souboru třídu `.round`s pravidem pro zakulacení na kolečko:
>   - `border-radius: 50%;`
> - Výsledek bude pak vypadat [takto](https://codepen.io/SimonB87/full/qBdMgKN)
