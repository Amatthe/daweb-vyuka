## Konfigurace live serveru

Nastavení Live Serveru v VS Code aby dokázal pracovat s routováním.

Obsah souboru `.vscode/settings.json`.

```json
{
  "liveServer.settings.root": "/dist",
  "liveServer.settings.file": "index.html"
}
```
